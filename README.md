# 🧮 Multi-Lang Calculator

A simple but clean console calculator implemented in **7 programming languages** – to demonstrate idiomatic code style and cross-language understanding.

## 🚀 Languages & run instructions

| Language | File | Run command |
|----------|------|--------------|
| C++ | `cpp/calculator.cpp` | `g++ calculator.cpp -o calc && ./calc` |
| Java | `java/Calculator.java` | `javac Calculator.java && java Calculator` |
| C# | `csharp/Calculator.cs` | `dotnet script Calculator.cs` or `csc Calculator.cs && mono Calculator.exe` |
| Go | `go/calculator.go` | `go run calculator.go` |
| Python | `python/calculator.py` | `python calculator.py` |
| JavaScript | `javascript/calculator.js` | `node calculator.js` |
| Rust | `rust/calculator.rs` | `rustc calculator.rs && ./calculator` |
